Предшаги:
1. Пользователь логинится
2. На вкладке "User Dashboard" нажимает Deposit Money
Ожидание: открыта вкладка Deposit Money с
- полями: Select Account (- Choose an account -), Enter Amount (пустое)
- кнопками: Deposit, Home, Logout


**тест 1. Позитивный тест, пользователь со счетами может положить деньги на депозит**
предусловия: у пользвателя есть счета
1) кликнуть в поле Select Account
ожидание: выпадающий список со счетами и балансами. Список соответствует
   GET' \'http://localhost:4111/api/v1/customer/accounts'
2) выбрать счет из списка
ожидание: выбранный счет отображается в поле Select Account
3) в поле Enter Amount ввести сумму 5000
4) нажать Deposit
   ожидание: алерт Successfully deposited $5000.00 to account {ACC number}!
после нажатия ОК алерта - открыта вкладка dashboard

выбранный в шаге 2 счет фактически пополнился на указанную сумму - проверка в 
'GET' \'http://localhost:4111/api/v1/customer/accounts'
