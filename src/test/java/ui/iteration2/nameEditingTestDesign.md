Предшаги:
1. Пользователь логинится
2. Кликает по Username или по name (если было задано) в правом верхнем углу рядом с Logout
3. Открыта вкладка Edit Profile c
-полем с плейсхолдером "Enter new name" если name не задано
или со значением name если задано
-кнопкой Save Changes


тест1. Пользователь успешно меняет name
1) в поле с плейсхолдером "Enter new name" ввести валидный name
ожидание: отображается введенное значение
2) нажать Save Changes
   ожидание: алерт "Name updated successfully!"
изменен name - проверка в 'GET' \'http://localhost:4111/api/v1/customer/profile
после закрытия алерта открыта та же вкладка Edit Profile,
в поле отображается значение из шага1
в правом верхнем углу должно отображаться новое name или после перезагрузки?

тест2. Пользователь не меняет name, если не нажимает Save Changes
1) в поле с плейсхолдером "Enter new name" ввести валидный name
   ожидание: отображается введенное значение
2) кликнуть вне поля
3) обновить страницу
ожидание: в поле отображается старое значение (либо плейсхолдер, если его не было)
в правом верхнем углу над username отображается старое name (либо ничего, если его не было)


тест3. Пользователь успешно меняет name. Проверка на отбрасывание пробелов в поле
1) в поле с плейсхолдером "Enter new name" ввести валидный name с пробелами в начале / конце строки
   ожидание: отображается введенное значение
2) нажать Save Changes
   ожидание: алерт "Name updated successfully!"
   изменен name - проверка в 'GET' \'http://localhost:4111/api/v1/customer/profile
   после закрытия алерта открыта та же вкладка Edit Profile,
   в поле отображается значение из шага1
   в правом верхнем углу должно отображаться новое name или после перезагрузки?
